# 胡桃生日会 · 雪霁梅香

一个为《原神》角色胡桃（往生堂第七十七代堂主）准备的生日会同人网站。

> 本网站为同人非官方作品，仅供交流学习使用。
> 本站使用的角色、音乐、图像等素材版权归原作者所有。

## 技术栈

- [Vite](https://vitejs.dev/) 6
- [React](https://react.dev/) 19
- [Tailwind CSS](https://tailwindcss.com/) 4
- [lucide-react](https://lucide.dev/)

## 本地开发

```bash
# 安装依赖
npm install

# 启动开发服务器
npm run dev

# 构建生产版本
npm run build

# 预览生产构建
npm run preview
```

开发服务器默认地址：http://localhost:5173

## 目录结构

```
public/          # 静态资源（图片等）
src/
  components/    # 可复用组件
  sections/      # 页面各板块
  data/          # 数据文件
  hooks/         # 自定义 Hooks
  lib/           # 工具函数
  theme.css      # 主题 CSS 变量
  index.css      # 全局样式
```
